## Usage
* 국내 모든 지번 (구 주소)를 담은 데이터 셋입니다. (2019년 6월 기준)
* 데이터는 시도, 구시군, 읍면동리로 세분화하여 JSON 형태로 저장되어있습니다. (`jibun_list.json`)
* 행정안전부에서 제공하는 주소 DB를 활용하였습니다.
